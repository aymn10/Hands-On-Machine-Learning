# PCA Dimensionality Reduction

## Overview

This project demonstrates Principal Component Analysis (PCA), a dimensionality reduction technique used to reduce high-dimensional datasets while preserving maximum information.

## Concepts Learned

- Dimensionality Reduction
- Principal Component Analysis (PCA)
- Feature Transformation
- Variance Preservation
- Data Visualization
- Standardization

## Libraries Used

- Pandas
- Matplotlib
- Seaborn
- Scikit-Learn

## Workflow

1. Generate dataset using `make_blobs()`
2. Standardize features
3. Apply PCA
4. Reduce dimensions from 5D to 2D
5. Visualize transformed data

## Key Takeaways

- Learned how PCA reduces complexity.
- Understood variance and principal components.
- Improved understanding of preprocessing pipelines.
- Learned how dimensionality reduction improves visualization and efficiency.
