# Ensemble Learning - Stacking Classifier

## Overview

This project demonstrates Stacking Ensemble Learning using multiple machine learning models combined into a single powerful classifier.

The Iris dataset is used for multi-class classification.

## Concepts Learned

- Ensemble Learning
- Stacking
- Meta Learning
- Decision Trees
- Support Vector Machines (SVM)
- Logistic Regression
- Classification Metrics

## Libraries Used

- Pandas
- NumPy
- Seaborn
- Scikit-Learn

## Workflow

1. Load Iris dataset
2. Encode target labels
3. Split dataset
4. Train base learners:
   - Decision Tree
   - SVM
   - Logistic Regression
5. Train meta learner
6. Create stacking classifier
7. Evaluate performance

## Key Takeaways

- Learned how ensemble models improve prediction accuracy.
- Understood the role of base learners and meta learners.
- Explored stacking architecture.
- Compared ensemble performance against individual models.
