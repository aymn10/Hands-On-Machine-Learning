# K-Means Clustering

## Overview
This project demonstrates the implementation of K-Means Clustering, one of the most widely used unsupervised machine learning algorithms.

The dataset is generated using Scikit-Learn's `make_blobs()` function and clustered into groups based on feature similarity.

## Concepts Learned

- Unsupervised Learning
- Clustering
- K-Means Algorithm
- Data Standardization
- Feature Scaling
- Cluster Centroids
- Elbow Method
- Inertia

## Libraries Used

- Pandas
- Matplotlib
- Seaborn
- Scikit-Learn

## Workflow

1. Generate synthetic dataset using `make_blobs()`
2. Scale features using `StandardScaler`
3. Apply K-Means clustering
4. Calculate inertia values
5. Use Elbow Method to determine optimal K
6. Visualize clusters and centroids

## Key Takeaways

- Learned how K-Means forms clusters.
- Understood the importance of feature scaling.
- Learned how the Elbow Method helps select the optimal number of clusters.
