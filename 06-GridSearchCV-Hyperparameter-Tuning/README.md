# Hyperparameter Tuning with GridSearchCV

## Overview

This project demonstrates how to optimize machine learning models using GridSearchCV.

The Iris dataset is used with K-Nearest Neighbors (KNN) to identify the best hyperparameter combination.

## Concepts Learned

- Hyperparameter Tuning
- Cross Validation
- Grid Search
- Model Optimization
- KNN Classification
- Train-Test Split
- Performance Evaluation

## Libraries Used

- NumPy
- Pandas
- Seaborn
- Matplotlib
- Scikit-Learn

## Workflow

1. Load Iris dataset
2. Split data into training and testing sets
3. Train KNN model
4. Define parameter grid
5. Apply GridSearchCV
6. Find best hyperparameters
7. Evaluate optimized model

## Key Takeaways

- Learned the difference between parameters and hyperparameters.
- Understood k-fold cross-validation.
- Learned how GridSearchCV automates model tuning.
- Improved model performance using optimal settings.
