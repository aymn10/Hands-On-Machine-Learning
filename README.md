# 🤖 Hands-On Machine Learning

A curated collection of hands-on Machine Learning projects built while learning fundamental to advanced ML concepts using Python and Scikit-Learn.

The goal of this repository is to strengthen practical understanding by implementing algorithms, experimenting with datasets, and exploring the complete machine learning workflow.

---

## 📚 Projects

| Project | Machine Learning Domain | Key Concepts Learned | Primary Algorithm |
|----------|-------------------------|----------------------|-------------------|
| Insurance Cost Prediction | Supervised Learning (Regression) | Data Preprocessing, Feature Engineering, Train-Test Split, Model Training, Performance Evaluation | Linear Regression |
| Heart Disease Prediction | Supervised Learning (Classification) | Binary Classification, Feature Scaling, Logistic Regression, Confusion Matrix, Accuracy Evaluation | Logistic Regression |
| Ford Car Price Prediction | Supervised Learning (Regression) | Exploratory Data Analysis (EDA), Feature Selection, Regression Modeling, Prediction | Linear Regression |
| K-Means Clustering | Unsupervised Learning | Clustering, Data Standardization, Elbow Method, Cluster Analysis | K-Means Clustering |
| PCA Dimensionality Reduction | Feature Engineering | Dimensionality Reduction, Principal Components, Variance Preservation, Data Visualization | Principal Component Analysis (PCA) |
| Hyperparameter Tuning | Model Optimization | Cross Validation, Hyperparameter Search, Model Selection, Performance Optimization | GridSearchCV |
| Ensemble Learning | Advanced Machine Learning | Ensemble Methods, Stacking, Meta Learning, Multiple Classifiers, Model Generalization | Stacking Classifier |
